# Historias de Usuario - Smart Clinic

1. **Doctor**: Como Doctor, quiero ver mi lista de citas del día para gestionar mis tiempos y brindar una atención organizada a mis pacientes.
2. **Paciente**: Como Paciente, quiero buscar doctores por especialidad y nombre para agendar una cita que se acomode a mis necesidades de salud.
3. **Administrador**: Como Administrador, quiero registrar nuevos doctores y gestionar la base de datos para asegurar que la clínica cuente con personal calificado.

### Funcionalidades Principales
- Registro de pacientes y doctores.
- Gestión de citas médicas en tiempo real.
- Generación de recetas médicas (Prescriptions).
- Consulta de disponibilidad de doctores.
